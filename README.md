# Pizza Shop API

## Description

This project was developed to learn Bun, Elysia and drizzle technologies! It`s a simple project to have Users, restaurants inside postgres DB and serving this data using the Elysia API.

## Technologies

- Bun
- Elysia
- Drizzle orm
- PostgresSQL

## Postgres

We create this docker container to run this Postgres DB.

Command to clear the docker compose files `docker-compose down --rmi all --volumes`.

### Commands

- `bun generate` -> Used to generate SQL files to be executed
- `bun generate` -> Used to generate SQL files to be executed
- `bun migrate` -> Used to apply the SQL files generated
- `bun studio` -> Used to visualize the Postgres data

## Bun

This project was created using `bun init` in bun v1.3.1. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

### Installing Bun

Check the bun guide to install it [Bun Installation guide](https://bun.com/docs/installation).

### Bun Commands

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev
```
